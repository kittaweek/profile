# เกี่ยวกับฉัน

ฉันเริ่มต้นจากนักพัฒนาซอฟต์แวร์ธรรมดา ทำเว็บไซต์ด้วย PHP กับ WordPress ก่อนจะขยายสู่ Laravel, Magento และยุค Modern JavaScript พร้อมกันนั้น ก็ได้ทำ Infrastructure ควบคู่กันมาตลอด จากการดูแล Linux Server และ Email System จนพัฒนาสู่ CI/CD Pipeline, Observability Stack และ Docker — ทำให้มองระบบได้ทั้งในมุม Developer และ DevOps เมื่อประสบการณ์สะสมมากพอ ฉันก็ได้ขึ้นมาเป็นหัวหน้าทีม เรียนรู้ทั้งเทคนิคการพัฒนาซอฟต์แวร์และการบริหารทีมไปพร้อมกัน

พอโลกเข้าสู่ยุค Data ฉันตัดสินใจเปลี่ยนเส้นทางไปสู่สาย Data Engineering และ Data Analysis เพื่อขยายมุมมองด้านข้อมูล แต่สุดท้ายพบว่าฉันชอบการพัฒนาซอฟต์แวร์มากกว่า จึงกลับมาพัฒนาซอฟต์แวร์เต็มตัวในฐานะ Software Engineer ที่ Connex Property และควบคู่กับการทำงาน ก็ได้ศึกษาและทดลองใช้ AI รวมถึงระบบ Automation พร้อมกับทบทวนความรู้ตัวเองอย่างต่อเนื่องมาตลอด

ความอยากรู้อยากเห็นและใจที่รักเทคโนโลยีคือแรงผลักดันหลักของฉัน ฉันถนัดการแก้ปัญหาเชิงตรรกะ มีมุมมองเปิดกว้าง และพร้อมปรับตัวเสมอ ทุกความท้าทายคือโอกาสเรียนรู้ — ไม่เคยรอให้พร้อมก่อนลงมือทำ ในเวลาว่างจึงมักลุยโปรเจกต์ส่วนตัวและเรียนคอร์สออนไลน์เพิ่มเติมอยู่ตลอด (ซึ่งตอนนี้มีโปรเจกต์ในหัวมีเยอะมากจนทำไม่ทันแล้ว!)

เมื่อไม่ได้อยู่หน้าจอคอมพิวเตอร์ ฉันชอบไปนั่งชิลที่ร้านกาแฟดี ๆ, ฟังพอดแคสต์หาความรู้ใหม่ ๆ, ออกไปวิ่ง, จัดทริปแคมปิ้งกลางธรรมชาติ หรือสนุกไปกับการเล่นบอร์ดเกมกับกลุ่มเพื่อน และเกม PC
