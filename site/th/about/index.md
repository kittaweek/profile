# เกี่ยวกับผม

ตอนนี้ผมทำงานเป็น ML Ops อยู่ที่ Data Café ครับ งานหลักๆ ก็คือ deploy, manage และ optimize machine learning models ใน production — ให้มัน work จริงๆ ไม่ใช่แค่ตอน dev

ก่อนหน้านี้ทำ Data Engineer อยู่ที่ Baania ดูแลข้อมูลอสังหาริมทรัพย์ทั่วไทย งานมัน complex และ scale ค่อนข้างใหญ่ ชอบครับ

แต่จริงๆ แล้วผมเริ่มมาจาก software engineer ทำ web application มาก็หลายปีก่อนจะมาทำ data ไม่ว่าจะเป็น PHP, WordPress, Magento 2, Node.js, TypeScript — ผ่านมาเกือบหมดแล้ว

## วิธีทำงาน

ชอบแก้ปัญหาครับ ยิ่ง problem มัน tricky ยิ่งสนุก ทำงานได้แบบ flexible และพยายาม improve ตัวเองอยู่เรื่อยๆ ทั้งหาคอร์สเรียนเพิ่มและ build side project ตามที่สนใจ
